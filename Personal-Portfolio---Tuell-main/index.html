Custom Computer Build

About This Project

This project is focused on the planning and the building of my own custom computer from the ground up. I started by putting all of my desired components into PCPartPicker, then I ordered the parts, all of which were on Amazon. Once all of the parts arrived, I assembled the system, installed Windows 11, and configured the UEFI settings for optimal performance.

Technical Details

  • Component Selection: Researched and chose compatible CPU, GPU, motherboard, RAM, storage, PSU and case.
  • System Assembly: Installed the motherboard, CPU + Cooler, RAM, storage, and GPU. Then, I wired power and data cables.
  • UEFI & OS Setup: Configured UEFI settings, set boot order, enabled XMP for the RAM, then installed Windows 11 + drivers.
  • Testing + Optimization: Ran stress tests with temperature monitoring, adjusted fan curves, and confirmed system stability under intense workloads.

Challenges & Solutions

The first challenge I ran into was before I even started building. The motherboard I ordered was damaged, so I had to return it and wait for a replacement. The second challenge I had was with the case's front panel connector. The case + motherboards' manuals were different in their instructions, but found out that I was installing it into the wrong port. I also ran into issues with the cables, since I did not know where certain cables went, primarily due to inexperience. It didn't help that the manual that came with the motherboard didn't label the cables, so I had to look up the online version to find where they went. The final issue I had during this journey was with updating the UEFI on the system. Since the motherboard's isntructions on how to do this were vague, I ended up doing it incorrectly. I had to re-flash it so that it could work.
